テンプレートのPowerPointを開き、テキストのみを今週の内容に差し替えて、新しい.pptxファイルを作成してください。
最終成果物はテキストの構成案ではなく、.pptxファイルです。

※本プロンプトは公開用のサンプルです。社名・競合名・配色はすべて架空（自社=Northwind）です。利用時は自社の実体に置き換えてください。

# 対象ファイル
- レポート: `~/Documents/competitor-watch/latest.md`
- テンプレート: `~/Documents/competitor-watch/slides/template.pptx`
- どちらかが存在しない場合は、生成せずに報告して終了（デザインをゼロから自作することは禁止）

# テンプレート方式の絶対ルール
- python-pptx でテンプレートを開き、**テキストの差し替えのみ**を行う
- 図形・カード・帯の追加、移動、サイズ変更、色変更、フォント変更はすべて禁止
- テキスト差し替えは run 単位で行い、元の書式（フォント・サイズ・色・太字）を維持する
- グループ化された図形の内部テキストも漏れなく対象にする
- テンプレートファイル自体は変更せず、別名で保存する

# 保存要件
- 保存先: `~/Documents/competitor-watch/slides/`
- ファイル名: `competitor_weekly_slides_YYYY-MM-DD_wXX.pptx`（latest.md の作成日・週番号に合わせる）

# スライド作成ルール
## 基本
- 5〜7枚（変化が乏しい週は3〜5枚目を統合し5枚まで圧縮可。タイトル・結論・アクションの3枚は必ず残す）
- 1スライド1メッセージ／見出しで結論を言い切る／抽象表現は禁止
## テキスト
- 箇条書きは1階層のみ／1行は全角28字以内／太字はキーワード1〜2箇所まで
- 文字数目安を超える場合はフォントを縮小せず文言を短くする

# スライドと差し替え内容
全スライド共通：フッターの日付・週番号を更新。
1. タイトル：メインタイトルは固定。右上スタンプ「ISO WEEK YYYY」の年と大型数字「WXX」の両方を更新。日付行は「YYYY年M月D日」。対象アカウント一覧。
2. 今週の結論：見出し（結論25字以内）／番号カード3枚（太字見出し28字＋補足55字）／「一言で言うと」帯（45字）
3. 市場の変化：見出し／変化3項目（小見出し＋本文）／新トレンド枠（社名＋見出し＋本文＋「→」示唆）
4. 競合の動き：見出し／数値カード3枚（社名／大きい数値8字／ラベル15字／本文60字）／「なぜ重要か」帯
5. 異常値：見出し／メイン異常値（大数値＋説明）／サブ2件／「読み取れること」
6. 自社への示唆：見出し／キーワードカード3枚（キーワード＋やること30字＋理由45字）／「追随不要」
7. アクションプラン：見出し／A-1・A-2（施策名25字／どの画面をどう変えるか60字／期待効果50字）／「判断いただきたい事項」または「次回確認」1行

# 可変要素
- 項目が3つに満たない場合は残りに「特筆すべき変化なし」（図形は削除しない。例外はスライド7のA-2のみ）
- 優先度A施策が1つの週はA-2カードを削除可。3つ以上は上位2つに絞り、残りは完了報告に記録（図形追加はしない）
- latest.md が初回ベースライン版の場合は、差分前提のスライドを現状整理に置き換える

# 作業手順
1. latest.md とテンプレートを読み込む
2. 対応表に従い各シェイプのテキストを差し替える
3. LibreOffice（soffice --headless）でPDF/画像に変換し、全スライドを目視確認する
4. テキストのはみ出し・差し替え漏れ（前週の内容が残っていないか）を確認し、あれば修正する
5. 保存し、完了報告（保存パス／各スライド見出し／短縮・省略した内容／見た目確認結果）を出力する
